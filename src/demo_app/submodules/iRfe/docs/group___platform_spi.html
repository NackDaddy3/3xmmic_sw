◆ PlatformSpi_readBurstAsync12()

sr_t PlatformSpi_readBurstAsync12 ( uint8_t  devId,
uint16_t *  bufRead,
uint32_t  count,
void(*)(void *)  cb,
void *  arg 
)

This function asynchronously reads captured 12bit sample data from the BGT60TRxx chip into 16bit buffer.

This function reads the sample data from the memory integrated in the BGT60TRxx chip with a SPI burst transfer.

This function allows to specify a callback function and associated data which will be called once the transfer is completed.

The whole burst transfer is performed in a single SPI frame, so the chip select signal (CS) goes LOW at the beginning and stays LOW until the end of the burst transfer.

Parameters
devIdSPI device Id
[out]bufReadA pointer to a buffer where received data
[in]countThe number of words to be transferred.
[in]cbFunction to call once the transfer has finished.
[in]argPointer to pass to the callback function.
