Additional Functions

GPIO Wrapper

Provides functionalities to interface the GPIO pins of the MMIC such as:

  • InitGpioWrapper : Sets the Pin Ids
  • Wrapper_GpioGet : Get the gpio state
  • Wrapper_GpioSet : Set the gpio to a state
  • Wrapper_GpioConfigure : Configure gpio
  • Wrapper_IsGpioConfigOutput : Determine whether the configuration is output

SPI Wrapper

Provide the wrapper to access the platform SPI functionalities:

  • InitSpiWrapper
  • Wrapper_SpiTransfer
  • Wrapper_SpiWrite
  • Wrapper_SpiConfigure
