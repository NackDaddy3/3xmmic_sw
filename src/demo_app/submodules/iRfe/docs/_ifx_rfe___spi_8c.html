error_t SpiTransfer ( uint32_t *  wbuf,
uint8_t  wlen,
uint32_t *  rbuf,
uint8_t *  rlen 
)

Transmit command and receive response in full duplex mode (send actual command and receive response from previous one)

Note
The spi frame is automatically extended if the response header indicates a longer frame than the command
The function is waiting for the RFT pin to be set, blockingly
Parameters
wbufData transmit
wlenNr. data words to transmit
rbufBuffer to receive into
rlen[out] Nr. of data words received
Return values
IFXRFE_E_SUCCESSif ok,
IFXRFE_E_CRC16on received header crc error
IFXRFE_E_CRC32on received frame crc32 error
IFXRFE_E_MISSING_PARAMETERif one of the parameters is NULL
IFXRFE_E_TIMEOUTif rft pin does not change in time (IFXRFE_RFT_TIMEOUT_DEF)
IFXRFE_E_UNEXPECTED_VALUEif received crc16 ok, but payload would not fit into buffer
Note
error code of the underlying platform spi layer otherwise
